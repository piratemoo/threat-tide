# vuln-feed
vuln-feed is a lightweight vulnerability feed that links directly to piratemoo.com/vuln.

It’s built to make it easier to quickly skim through newly disclosed CVEs, public PoCs, exploit writeups, and useful security research without digging through a bunch of noise.

- Recent CVEs
- Public PoC references
- Severity indicators
- Researcher attribution
- Tags and exploit context
- Simple feed-style UI

The project is meant to be fast, clean, and easy to browse. Just open the feed, skim what’s new, and decide what’s worth investigating further.
